## 📋 Tentang Website

Website resmi IP2TP Batang adalah platform digital yang menghubungkan riset pertanian, teknologi, dan petani untuk mendukung ketahanan pangan.

### Fitur Utama
- 📊 Dashboard Pertanian Real-time
- 💬 Chatbot TaniBot AI
- 📅 Kalender Tanam Digital
- 🐛 Laporan Hama Online
- 🗺️ Peta Digital Lahan (SILEPBAH)
- 📂 Open Data Pertanian

## 🛠️ Teknologi
- PHP 7.4+
- Tailwind CSS
- Chart.js
- JavaScript
